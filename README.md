# Survival Island - Mini Game (C)

Desenvolvido como trabalho acadêmico por Alvaro Vogas.

Este projeto implementa um mini-game em C com três camadas principais:
1. Inventário (mochila) - versão com vetor e versão com lista encadeada.
2. Simulação de ataques de recursos (não-violento): operações de remover/transferir itens (simplificado).
3. Módulo de componentes para montagem da torre de fuga com algoritmos de ordenação e busca binária.

## Como compilar
No diretório do projeto execute:

```bash
make
```

## Como executar
```bash
./survival
```

Siga o menu interativo.

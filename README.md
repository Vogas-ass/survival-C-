Survival Island - Mini Game em C
================================

Estrutura do repositório:
- include/  -> headers (item.h, array_inventory.h, list_inventory.h, utils.h, components.h, sorts.h)
- src/      -> fontes .c (main.c, array_inventory.c, list_inventory.c, utils.c, components.c, sorts.c)
- Makefile  -> compilar com 'make'

Como compilar:
  make

Como executar:
  ./survival

Descrição:
Projeto para fins acadêmicos que implementa um inventário por vetor e por lista encadeada,
operações básicas (inserir, remover, listar, buscar), ordenações e um módulo simples de componentes.
